# Telegram-бот по «Капитанской дочке»

Это Telegram-бот для разбора «Капитанской дочки» через диалог и короткие вопросы.
Он помогает читать историю по шагам, отвечать в своём темпе и лучше понимать героев, события и смысл сцен.

## Как работает бот

Пользователь читает небольшой фрагмент, отвечает на вопрос и получает короткую поддерживающую реакцию.
Бот не давит, а мягко помогает заметить важное и ведёт дальше по истории.

## Этапы

1. Герои и сюжет — знакомство с героями и основными событиями.
2. Пересказ — более внимательный разбор той же истории по эпизодам.
3. С цитатами — более точное понимание текста с опорой на формулировки из произведения.

## UX-особенности

- Бот не проверяет, а помогает разбираться.
- Почти любой осмысленный ответ принимается.
- В диалоге нет давления и ощущения экзамена.
- Можно написать `стоп`, чтобы пропустить вопрос и перейти дальше.

## Технически

- Бот работает через Telegram Bot API.
- Логика написана на Python.
- Есть state для отслеживания прогресса пользователя.
- Поддерживаются вопросы с вариантами ответа: `1`, `2`, `3` или текст варианта целиком.

## Запуск

```bash
pip install -r requirements.txt
python bot.py
```

## Переменные окружения

В `.env` нужно указать:

```env
BOT_TOKEN=...
OPENAI_API_KEY=...
```
